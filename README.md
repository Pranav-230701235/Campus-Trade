# Campus Trade REC 🎓
A marketplace app specifically designed for the **Rajalakshmi Engineering College** community.

### ✨ Key Features
* **Firebase Auth:** Login restricted to college email domains.
* **Firestore Database:** Real-time request wall and product listings.
* **FCM Notifications:** Automated alerts for new student requests.

### 🛠️ How to Run
1. Clone this repository.
2. Add your own `google-services.json` from the Firebase Console into the `/app` folder.
3. Build and run in Android Studio.
